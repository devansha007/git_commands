lets know about VCS

list of commands

  • git init : to initialise git
  • git add . : to add all data to the staging area
  • git commit - m "your custom message" : to add the changes permanantly
  • git remote add origin "http//your code copy website" : to add the changges to the cloud
  • git reset --hard {your log} : to reset the changes after the commit
  • git clone : to clone a repo
  • git pull : to update the cloned repo
  • git push : it is used to push the changes in the remote repo from local repo
  • git branch " " : to create a branch
  • git checkout " " : to change a branch
  • git merge : to merge the changes in the diferent branch and this branch
